# Sales Data ETL Pipeline Project

## Overview
This project demonstrates an end-to-end ETL pipeline using sales data.

## Dataset
- accounts.csv
- products.csv
- sales_pipeline.csv
- sales_teams.csv
- data_dictionary.csv

## Objective
To process raw sales data and transform it into analytics-ready format.

## Workflow
1. Data ingestion from CSV files
2. Data cleaning (remove nulls, duplicates)
3. Data transformation (joins, aggregations)
4. Output to structured format (Parquet/CSV)

## Technologies Used
- Python
- PySpark
- SQL

## Sample Use Case
- Sales performance analysis
- Product-wise revenue tracking
- Team performance insights

## Future Enhancements
- Integrate with Azure Data Factory
- Load data into Data Lake (ADLS)
- Build dashboards using Power BI
